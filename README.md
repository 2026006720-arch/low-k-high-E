# low-k-high-E
Concurrent dielectric and mechanical property prediction of low-k materials via machine learning.
This repository contains the code and data for the physics-constrained multi-task machine learning framework described in our manuscript. 
Repository Structure:
* Training Model Code and Training Dataset: Python implementation of the XGBoost multi-task framework, Optuna optimization, and the training dataset.
* Model Screening and Dataset: Scripts and datasets for high-throughput screening of 729 candidate materials.
* VASP Validation Details: INCAR, KPOINTS, and POSCAR files for the DFT validation of SiO_2, Mg_3Si_4(HO_6)_2, and B_2S_2O_9.
